# K League (가칭) MVP 와이어프레임 v0.1

작성: 2026-04-15
기반: 기획문서 v0.2, API-Football 스키마 실증 결과.

> ASCII 목업은 모바일 세로 화면 기준 (폭 40자 ≈ 390pt). 최종 디자인 아닌 **데이터 배치·정보 위계·네비게이션 검증용 초안**.

---

## 0. 정보 구조 (IA)

```
Tab Bar (하단 5탭)
├── 🏠 홈 (대시보드)
├── 📅 일정
├── ⚽ 경기   ← (옵션) 오늘 경기만 보는 빠른 탭
├── 🌍 해외파
└── 👤 마이
```

**대안**: 탭 4개로 줄이고 ⚽ 제거 → 경기는 일정 탭에서 접근. 직관러 페르소나엔 4탭이 깔끔.

### 주요 스크린 진입 경로
| 목적지 | 진입 경로 |
|---|---|
| 경기 상세 | 홈 → 다음경기 카드 · 일정 → 경기행 · 팀 → 최근/다음 경기 |
| 팀 상세 | 홈 → 내 팀 · 일정 → 팀명 · 순위 → 팀명 |
| 선수 상세 | 팀 → 스쿼드 · 경기 → 라인업 · 득점 순위 → 선수명 |
| 경기장 정보 | 경기 상세 → 장소 탭 · 팀 → 홈구장 |
| 직관 체크인 | 경기 상세 → 체크인 버튼 (경기 당일 표시) |

### 데이터 매핑
| UI 요소 | API 엔드포인트 · 필드 |
|---|---|
| 내 팀 로고·컬러 | `/teams?id=X` → `team.logo`, 컬러는 자체 매핑 |
| 카운트다운·킥오프 시간 | `/fixtures?team=X&next=1` → `fixture.date` + `fixture.timestamp` (KST 변환) |
| 경기장·도시 | `fixture.venue.{name, city}` |
| 라운드 | `league.round` |
| 최근 결과 | `/fixtures?team=X&last=1` → `goals.home/away` + `teams` |
| 골 정보 | `/fixtures/events?fixture=X` (타입=Goal) |
| 순위 | `/standings?league=292&season=Y` |
| 해외파 섹션 | 유럽파 선수 ID 자체 관리 → `/players?id=X&season=Y` 집계 |

### 상태 / 엣지 케이스
- 다음 경기 없음 (시즌 오프) → "시즌 종료 · 프리시즌 소식" 대체 카드
- 경기 당일 → 카운트다운 대신 "오늘 · 19:00 킥오프" + 경기 시작 시 LIVE 배지
- 경기 중 → "LIVE 1-0 · 67'" 스코어 자동 갱신 (v1.1)
- 내 팀 미선택 → 온보딩 리디렉트

> (출처: `docs/data-source-validation.md` §4.3).

### 데이터 매핑
- 상단 스코어 블록: `/fixtures?id=X` → `goals`, `score.halftime/fulltime`, `teams`, `fixture.venue`, `fixture.referee`
- 방송 · 중계 링크: 자체 매핑 테이블 (K1 기준 쿠팡플레이·TVING 중계권 매핑 필요 — 시즌 초 연맹 공식 공지 기반 갱신). 네이버 문자중계는 `https://sports.naver.com/game/{gameId}` 형식 딥링크
- 하이라이트: K리그 공식 유튜브 채널 API (YouTube Data API v3) or RSS 파싱으로 fixture_id ↔ video_id 매핑 테이블 구축
- 주요 장면: `/fixtures/events?fixture=X` 중 Goal·Red 카드만 필터
- 타임라인(탭 4): `/fixtures/events?fixture=X` 전체 (정렬 by `time.elapsed`)
- 라인업: `/fixtures/lineups?fixture=X` → `formation`, `startXI`, `substitutes`, `coach`
- 포메이션 뷰: `startXI[].player.grid` ("row:col" 형식) 기반 SVG 렌더
- 장소: `fixture.venue.{name, city}` + 자체 보강 테이블 (주소·교통·주차·좌석배치도 수기 큐레이션)

### 상태 / 엣지 케이스
- 경기 전 (NS) → 장소·방송 섹션 우선 노출, 라인업은 "킥오프 1시간 전 공개" 빈상태
- 라이브 중 → 상단 스코어·타임라인 실시간 폴링 (v1.1), 하이라이트 숨김
- 종료 후 → 하이라이트 영상 자동 노출, 중계 링크는 "다시보기" 라벨로 변경
- 연장/승부차기 (AET/PEN) → `score.extratime`, `score.penalty` 추가 블록
- 하이라이트 영상 미발행 → 섹션 자체 숨김 ("데이터 없음" 배너 지양)
- 직관 체크인 → 경기 당일(로컬 시간 ±6h)에만 버튼 활성화

- 시즌 오프 / 부상 → "부상 결장" 배지
- 이적 시즌 → 팀 변경 감지 배치 잡 필요
- API 누락 선수 → 수동 편집 가능한 관리자 도구 (운영 이슈)
