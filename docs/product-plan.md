# K리그 앱 — 제품 기획 문서 (v0.2)

## Context
- **왜 만드는가**: 지인이 K리그 열성 팬인데 현재 나와있는 앱/서비스가 만족스럽지 않다고 이야기함. 그 니즈를 해결하는 것이 출발점.
- **작성자 배경**: 한국 축구 도메인 지식이 거의 없음 → 이 문서는 "뭘 만들지"를 구조화해서 **지인에게 직접 들어야 할 질문**을 뽑아내는 용도로도 활용.
- **단계**: 기술 제약 없는 **제품 기획** 단계. 스택/배포/API 선택은 이후 단계.
- **결과물**: 경쟁 앱 대비 차별점, 핵심 기능 MVP, 확장 기능 로드맵, 풀어야 할 오픈 이슈 정리.

## 0. 초기 방향성 결정 (Q&A 반영)
사용자가 확정한 전제:
- **타겟 페르소나**: **직관러** (경기장 자주 가는 팬). 따라서 오프라인 직관 보조가 킬러 기능.
- **커버 범위**: K리그1/2 + FA컵·ACL + **국가대표·해외파(손흥민·이강인 등)까지**. 스코프 큼.
- **성격**: **실서비스 지향 (스토어 출시)**. 품질·안정성·법적 검토·운영 체계 필요. 네이티브 앱 우선.

→ **전략 포지셔닝**: **"C(직관 컴패니언) 중심 + A(내 팀 대시보드) 결합"**
→ 차별화 1순위: 직관 경험 통합. 차별화 2순위: 한국 팬이 실제로 챙겨보는 범위(K리그+국대+해외파) 한 앱에.

## 1. K리그 생태계 한눈에 (도메인 배경)

지인이 어떤 스코프를 원할지 정하려면 먼저 K리그 구조부터 알아야 함.

### 국내 대회
- **K리그1** (1부): 2026 시즌 12팀, 28주 일정 + 스플릿(파이널A/B) — 2026이 **스플릿 마지막 시즌**. 2027부터 14팀, 3라운드 로빈.
- **K리그2** (2부): 2026 시즌 **17팀으로 확장**. 우승·준우승 자동 승격, 3~6위 플레이오프.
- **승강 플레이오프**: K1 최하위권 ↔ K2 상위권.
- **Korea Cup (FA컵)**: 2026부터 **가을-봄 포맷 전환**. 2026년 7~8월 개막 → 2027년 5~6월 결승. 아마추어·세미프로도 참가.
- **R리그 / U18 / U17 주니어 리그**: 유소년·리저브. (공식 앱만 커버)

### 국제 대회 (K리그 클럽 관점)
- **AFC Champions League Elite** (ACLE): 아시아 최상위 클럽 대항전. K1 상위권 진출.
- **AFC Champions League Two** (ACL2): 2부 대회.

### 관련 레이어 (앱 스코프 확장시 고려)
- **국가대표팀** (A매치, U-23, 여자), **해외파** (손흥민·이강인류), **WK리그** (여자 프로), **K3/K4** (세미프로).

### 구단 (2026 K1)
울산, 전북, 포항, 김천 상무, FC서울, 광주, 대구, 수원FC, 제주, 인천, 강원, 대전 하나 — 참고용.

## 2. 타겟 유저 & 시장 상황

### 가정 페르소나
| 페르소나 | 행동 특징 | 주요 니즈 |
|---|---|---|
| **직관러** | 홈경기 시즌권, 원정 직관 가는 편 | 티켓, 교통/주차, 굿즈, 응원가, 직관 후기 |
| **TV/중계러** | 쿠팡플레이·TVING에서 경기 시청, 실시간 반응 | 라이브스코어, 문자중계, 하이라이트, 방송 링크 |
| **분석러** | 선수 스탯·전술 파고듦 | 상세 통계, 히트맵, 포지션별 기록, 역대 기록 |
| **커뮤니티러** | 에펨코리아·디시·더쿠 상주 | 글 피드, 반응/댓글, 밈, 팬 라디오/방송 |
| **라이트 팬** | 좋아하는 팀 결과·일정만 가볍게 | 내 팀 위젯, 푸시 알림, 순위표 |

### 경쟁 앱 요약
| 앱 | 포지션 | 강점 | 약점 |
|---|---|---|---|
| **K리그 공식 앱** | 연맹 공식 | 전 대회 커버, 티켓, 데이터센터 | UX 올드, 멤버십/마케팅 섞임, 앱 무거움 |
| **Kick** (공식 멤버십) | 팬 활동 포인트 | 출석·직관 인증, 파트너사 혜택 | 정보 조회 기능 약함 (멤버십 중심) |
| **오늘의 K리그** (야마스튜디오) | 써드파티 정보 앱 | K리그 포커스, 구단 SNS 연결 | 앱 튕김, 디자인 구세대, 리뉴얼 예정 |
| **네이버 스포츠** | 포털 | 접근성, 문자중계, 오픈톡 | 국대/1부/2부 **정보가 섞여 탐색 어려움**, 중계권 없음 |
| **FotMob / Flashscore** | 글로벌 | UX 준수, 라이브스코어 강함 | K리그 깊이 얕음, 한국어 컨텍스트 부족 |
| **에펨코리아·디시·더쿠** | 커뮤니티 | 정보·반응 풍부 | 파편화, 비팬엔 진입장벽 |

### 시장 공백
1. **"K리그만" 깊게 다루는 모던 앱**이 약함 — 공식 앱은 올드하고, 써드파티는 불안정하거나 글로벌 앱은 얕음.
2. **커뮤니티 반응이 분산**돼 있음 — 정보와 반응이 서로 다른 플랫폼.
3. **개인화(내 팀 중심 뷰)**가 약함 — 관심 팀만 깔끔하게 보는 경험 부족.
4. **직관러 실용 기능** (경기장 정보, 교통, 응원석, 굿즈) 통합이 없음.

## 3. 제품 포지셔닝 (가설)

> **"내 K리그 팀만 깔끔하게, 한 앱에서."**
> 공식 앱처럼 모든 걸 담지 않고, **라이트~코어 팬이 매일 열어보는 대시보드**가 되는 것에 집중.

3가지 전략 옵션 (택1 or 조합):

**A. 내 팀 중심 대시보드 (라이트·코어 팬)**
- 관심 팀 등록 → 일정·결과·뉴스·라인업 알림 통합
- UX 미니멀, 위젯/푸시 중심

**B. 분석·스탯 덕후 앱**
- K리그 공식 데이터포털(data.kleague.com)·써드파티 통계 재가공
- 선수 비교, 시즌별 추이, 인포그래픽

**C. 직관 컴패니언**
- 경기장·좌석·교통·굿즈·응원가·체크인
- 오프라인 직관 경험 보조

> 가장 넓게 먹히는 건 **A**, 차별화 강한 건 **B 또는 C**. 지인이 어느 페르소나인지에 따라 포커스 다름.

## 5. MVP 제안 (v1.0 — 실서비스 출시 대상, 8~12주 개발 가정)

**"직관러를 위한 내 팀 컴패니언"** 컨셉. 직관 보조 + 기본 정보 대시보드를 한 앱에.

### 포함 (v1.0)
1. **응원팀 선택 & 온보딩** — K1/K2 복수 팀 가능, 국가대표 기본 구독
2. **홈 대시보드** — 내 팀 다음 경기 카운트다운, 최근 결과, 순위 요약
3. **경기 일정·결과** (K1/K2 + FA컵 + ACL + A매치, 리그·팀별 필터)
4. **경기 상세** — 라인업, 스코어, 득점자, 주요 이벤트, 경기장 정보, **방송사 중계 링크(쿠팡플레이·TVING·네이버 문자중계)**
5. **순위표** (K1/K2, 홈/원정 탭)
6. **팀 상세** (스쿼드, 최근 5경기, 다음 경기, 홈 경기장, **구단 공식 SNS 바로가기 — 인스타·유튜브·X**)
7. **선수 상세** (기본 프로필 + 시즌 스탯)
8. **푸시 알림** — 킥오프 1시간·10분 전, 골/종료 스코어, 라인업 공개
9. **홈·잠금화면 위젯** — 다음 경기, 최근 결과 (실서비스 차별점)
10. **경기장 정보 카드** — 주소, 대중교통, 주차, 좌석 배치도, 티켓 예매 **외부 링크**
11. **직관 체크인 & 기록** — 경기 체크인 → "이번 시즌 N경기 직관" 통계
12. **뉴스 피드** — K리그 + 국대 + 해외파(손·이강인 포함), 매체·팀별 필터
13. **해외파 탭** — 유럽 리그 소속 한국 선수 일정·결과·최근 경기 요약
14. **하이라이트 영상** — 경기 상세 [요약]에 K리그 공식 유튜브 임베드 (경기 종료 후)

### 제외 (v1.x 이후)
- 라이브 문자 중계 (복잡, 데이터 소스·제휴 필요) → v1.1
- 커뮤니티·채팅·댓글 → v2.0 (법적·운영 리스크 큼)
- **경기 상세 '통계' 탭** (xG·점유율·슈팅) — 무료 검증에서 데이터 미확보 → v1.1 (유료 플랜 전환 후 재검증)
- 분석·히트맵 → 데이터 소스 확보 후
- 판타지·예측 → v2.0+
- WK리그·K3·K4 → 코어 팬 피드백 본 뒤

### v1.1 ~ v2.0 로드맵
1. **라이브 스코어 & 문자 중계** (직관 못 가는 날 니즈)
2. **경기 상세 통계 탭** (xG·점유율·슈팅·유효슈팅 등 — 유료 플랜 전환 후 재검증 필수)
3. **굿즈샵 링크 / 응원가·구호 가이드** (직관 보조 심화)
4. **H2H / 상대전적**
5. **국대 A매치 전용 모드** (월드컵·아시안컵 기간 활성화)
6. **선수 비교 / 스탯 인포그래픽**
7. **공식 티켓 연동** (가능하면 제휴)
8. **판타지 / 예측 게임**
9. **팬 커뮤니티** (운영 체계 갖춘 뒤)

## 6. 데이터 소스 (실서비스 관점 — 안정성·법적 리스크 고려)

> 실서비스 출시하려면 **데이터 라이선스 명확성**이 핵심. 애플/구글 스토어 심사와 향후 법적 분쟁 대비.

| 소스 | 커버 | 비용 | 안정성 | 법적 |
|---|---|---|---|---|
| **API-Football (api-sports.io)** | K1/K2, 해외 리그(손·이강인 포함), A매치 | 무료 100콜/일, 유료 플랜 적극 필요 | ○ | ○ (상업 이용 가능, 플랜 확인) |
| **Sportmonks** | 글로벌, 한국 리그 포함 | 유료 | ◎ | ◎ |
| **Sportradar** | 최상위 | 매우 비쌈 | ◎ | ◎ |
| **Live-Score-API** | K1/K2/K-Cup/K3 | 저렴 | ○ | ○ |
| **K리그 공식 데이터 포털** (data.kleague.com) | K리그 공식 | 미확인 — **연맹 제휴 문의 필요** | ◎ | ◎ (정식 제휴시) |
| **웹 스크래핑** (공식·네이버) | 전체 | 무료지만 리스크 | △ 구조 변경·차단 | ✗ 약관 위반 소지, 실서비스 부적합 |
| **비공식 API** (FotMob·Sofascore 내부) | 전체 | 무료지만 회색지대 | △ | ✗ 실서비스 부적합 |
| **구단 공식 RSS·유튜브·SNS** | 뉴스·하이라이트 | 무료 | ○ | ○ (임베드 기준) |

**권장 조합 (실서비스)**
1. **유료 API 1개 구독**: API-Football 유료 플랜 or Sportmonks. 글로벌 커버라 국대·해외파 함께 처리 가능.
2. **연맹·구단 제휴 탐색**: 실서비스라면 장기적으로 **K리그 연맹·KFA와의 공식 제휴 또는 데이터 라이선스 문의**가 가장 안전한 경로.
3. **뉴스는 RSS 집계**: 스포츠조선, 스포츠서울, 인터풋볼 등 매체 RSS + 구단 공식 계정.
4. **하이라이트**: K리그 공식 유튜브 임베드(퍼블릭 허용 범위).
5. **절대 금지**: 경쟁 앱 내부 API 역공학, 스크래핑 기반 상용 서비스.

## 7. 플랫폼 옵션 (실서비스 기준)

직관러 페르소나 + 실서비스 + 위젯·푸시·체크인 필수 → **모바일 앱이 사실상 확정**.

| 옵션 | 평가 |
|---|---|
| **iOS/Android 네이티브 (Swift·Kotlin)** | 위젯·푸시·체크인 품질 최상. 개발·유지보수 비용 2배. |
| **Flutter** | 한 코드 베이스, 위젯·푸시 지원도 좋아짐. 실서비스 사례 많음. **추천 1순위**. |
| **React Native** | 생태계 크지만 위젯은 네이티브 모듈 필요. |
| **웹/PWA** | 실서비스엔 부적합 (스토어 출시 제외되고, 푸시·위젯 한계). |
| **웹 프로토타입 → 앱** | 데이터·UX 검증 단계에선 유효. |

**권장 경로**: **웹 프로토타입(1~2주)로 데이터·UX 검증 → Flutter 앱으로 v1.0 출시**. 또는 초기부터 Flutter 직행.

- **운영 부담**: 뉴스 큐레이션·푸시 정확도·데이터 정확성 민원 → **CS 채널**·유지보수 인력·비용.
