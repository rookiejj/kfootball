## 한 줄 결론

> **우리가 만들고 싶은 앱의 거의 모든 화면을 실제로 만들 수 있습니다.
> 단, 경기 정보를 받아오는 외부 서비스에 매달 구독료를 내야 합니다.**

---

## 왜 이런 걸 조사했나

K리그 앱을 만들려면 여러 정보가 필요해요:
- 오늘 누가 어디서 몇 시에 경기하는지
- 현재 순위는 어떤지
- 선수 프로필과 기록
- 어제 경기 득점자와 스코어

그런데 **K리그 연맹이 앱 개발자에게 이런 데이터를 직접 나눠주는 창구를 따로 열어두지 않았어요.** 그래서 "전 세계 축구 데이터를 모아서 파는 외부 서비스"를 이용해야 합니다.

그 중 가장 많이 쓰이는 `API-Football`이라는 곳이 **K리그 데이터를 제대로 주는지, 우리가 그리려는 화면 하나하나가 실제로 만들어질 수 있는지** 직접 테스트해봤습니다.

## 왜 무료로는 안 되고 유료여야 하나

외부 서비스에 **무료로 가입**해서도 테스트해봤어요. 결과는:

| | 무료 계정 | 유료 계정 |
|---|---|---|
| **올해(2026) 경기 정보** | ❌ **잠겨 있음** | ✅ 열려 있음 |
| **"앞으로 있을 경기" 조회** | ❌ **차단됨** | ✅ 됨 |
| 과거 시즌 | ✅ 2022~2024만 | ✅ 전부 |
| 하루에 요청 가능한 횟수 | 100번 | 수천~수만 번 |

**무료로는 "오늘 울산 경기가 몇 시에 시작하나?"라는 가장 기본적인 질문에 답할 수가 없어요.** 사용자가 수백 명만 되어도 100번 한도는 바로 넘기고요.

그래서 **유료 구독은 전제조건**입니다. 정확한 월 요금은 가입 직전에 한 번 더 확인합니다.

| 대안 | 왜 안 쓰기로 했는가 |
|---|---|
| 비슷한 외부 서비스 (Sportmonks 등) | **무료 플랜이 K리그를 아예 주지 않음.** 유료 가격 비교는 결제 직전에 한 번 더 |
| K리그 공식 홈페이지에서 자동으로 긁어오기 | 약관 위반 소지가 있어 상업용 앱엔 부적합 |
| 다른 축구 앱이 쓰는 데이터를 빼오기 | 법적 위험 큼 |
| K리그 연맹과 공식 제휴 | 가장 좋은 루트지만 답변 오래 걸림. 일단 유료로 시작하고 제휴 문의는 병행 |

## 결국 어떻게 하기로 했나

1. **외부 서비스 `API-Football` 유료 구독을 기본 데이터 공급원으로 사용한다.**
2. **첫 출시 버전에 넣을 14개 기능은 모두 구현 가능.** 그중 5개 정도(경기장 주소·구단 SNS·하이라이트·뉴스·중계 링크)는 우리가 직접 관리할 목록이 조금 있음.
3. **경기별 점유율·xG 같은 고급 통계는 다음 업데이트**로 미루고, 유료 가입 직후 한 번 더 확인한다.
4. **K리그 연맹과의 공식 제휴 문의는 별도로 넣어둔다.** 답이 오면 더 깨끗한 경로로 전환 가능.
